markdown# Terminal Messenger

Чат в терминале на Java. Можно общаться в каналах и писать личные сообщения.

## Запуск

Сначала сервер:
```bash
cd Server/src
javac Server.java
java Server
```

Потом клиент (можно открыть несколько):
```bash
cd Client/src
javac Client.java
java Client
```

## Команды
/register <имя> <пароль>   зарегистрироваться
/login <имя> <пароль>      войти
/join <канал>              выбрать канал — general, tech или random
/dm <имя> <текст>          написать личное сообщение
/history <имя>             посмотреть историю личной переписки
/users                     кто сейчас онлайн
/quit                      выйти
просто текст               отправить сообщение в канал

## Что умеет

- регистрация и вход, данные хранятся в файле
- три канала для общения
- личные сообщения между пользователями
- история сообщений за последние 3 дня подгружается при входе в канал
